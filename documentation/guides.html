ChromaWallet Documentation

Installation Guides

Windows

  1. Download and install the latest bitcoin-qt client from here.

  2. Navigate to your %APPDATA%\Bitcoin directory.

  3. Create a text file entitled bitcoin.conf. In this file, place the following text, substituting a password of your choosing in the rpcpassword field.

    server=1
    rpcuser=rpcuser
    rpcpassword=yourpassword

  4. Open a command window and navigate to your Bitcoin installation directory.

  5. Setup bitcoin-qt to run on the testnet with txindex enabled with the following command:

    bitcoin-qt.exe -testnet -txindex

    (Alternatively, run bitcoind.exe with same options.) Note: If you previously ran either bitcoin-qt of bitcoind without the txindex flag, you need to use the reindex option one time only.

  6. Download the latest Windows version of ChromaWallet from here, extract it to a location of your choosing, and navigate a command window to the ngccc directory.

  7. You can now use either the command-line version of ngccc with this command:

    ngccc-cli--help

    OR the GUI version with this command:

    ngccc-gui

Linux

  1. Download and install the latest bitcoin-qt executable file from here, or via your package manager

  2. Navigate to your ~/.bitcoin/ directory, and create a text file entitled bitcoin.conf. In this file, place the following text, substituting a password of your choosing, in the rpcpassword field.

    server=1
    rpcuser=rpcuser
    rpcpassword=yourpassword

  3. Setup bitcoin-qt to run on the testnet with the following command and wait for it to reindex the blocks.

    bitcoin-qt.exe -testnet -txindex -reindex

    OR setup bitcoind to run on the testnet with the following command and wait for it to reindex the blocks.

    bitcoind.exe -testnet -txindex -reindex -daemon

    Note: If you previously ran either bitcoin-qt of bitcoind without the txindex flag, you need to use the reindex option one time only.

  4. Download the latest Linux version of ChromaWallet from here, extract it to a location of your choosing, and open a terminal window in that location.

  5. You can now use either the command-line version of ngccc with this command:

    ./ngccc-cli--help

    OR the GUI version with this command:

    ./ngccc-gui

Running Multiple Clients on One Machine

  1. If you have already ran ChromaWallet, you will simply need to copy the ngccc directory, delete the wallet.db in the copied directory, and then run ngccc-cli or ngccc-gui from the new directory. A new wallet will be created at runtime

  2. Otherwise, download and extract the latest version of ChromaWallet as described in the previous sections.

  3. Copy the ngccc directory.

  4. Run ngccc-cli or ngccc-gui from the original directory and also from the copied directory.

  5. New wallets will be created at runtime. You can now send bitcoins and assets between the two wallets

Asset Management

Issuing an Asset

  1. Click on the asset tab, and then click on the issue new asset button.

  2. Enter a moniker of your choosing, select a color scheme, quantity, and number of units (satoshis) and then click okay.

  3. The asset should appear in your asset tab. If you go to the overview tab, your asset should be listed along with its balance and address. Note, you may have to restart the client for the balance to update. This is a known bug.

Adding an Existing Asset

Method One: Import Asset via JSON (Recommended)

  1. Click on the asset tab in a client that already contains the asset you wish to copy.

  2. Right click on the asset you wish to copy, and select copy as JSON.

  3. In the client that you wish to contain the new asset, click on the asset tab, and then click on the Import asset from JSON button.

  4. Right click in the box, paste the JSON that you previously copied, and then click OK. Your asset should now be listed.

Method Two: Manual Addition

  1. Click on the asset tab, and then click on the add existing asset button.

  2. Enter the moniker, color description, and unit of the asset you want add.

  3. The asset should appear in your asset tab and in your overview tab. You can now send coins via the send tab just as you would send regular bitcoins. Note that there is a dropdown box on the send coins tab to select bitcoins or an asset that your wallet contains. Please ensure that you select the proper one to send.

P2P Trade

  1. Click on the P2P Trade Tab.

  2. Select the asset you wish to buy or sell.

  3. Enter the quantity and price of the asset you wish to buy or sell, and then click the buy or sell button. Delete a buy or sell listing by double clicking on the listing.

  4. When a buyer or seller is found to match your listing, a trade will occur. Note, advanced matchmaking has not been implemented yet, but it will be implemented in future releases.
